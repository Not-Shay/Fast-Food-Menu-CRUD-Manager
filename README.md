# Fast Food Menu Manager 

A Python-based CRUD application for managing restaurant menu items. This project uses **SQLite** for persistent data storage.

## Features
- **Create:** Add new items (Burgers, Drinks, etc.) to the menu.
- **Read:** View the full menu sorted by Category and Name.
- **Update:** Change prices of existing items safely.
- **Delete:** Remove items from the menu.
- **Crash Proof:** Handles invalid user inputs (like typing text into price fields) without crashing.

## Technologies Used
- Python 3.13
- SQLite3 (Built-in)
